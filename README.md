使用云开发  与 vant

* 云开发
创建云开发 并在app.js 中替换 env
创建数据库 名称为 hblist
将云开发代码部署到 云服务器
将cloud下 json.json 导入到hblist云数据库中

* vant 
构建npm 具体参考vant

* 其他
请开启 增强编译  es6转es5 使用npm模块
